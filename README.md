# PrivacyLens GitHub Pages Privacy Policy

This folder contains a static privacy policy page for PrivacyLens.

## Upload To GitHub

1. Create a GitHub repository, for example `privacylens-policy`.
2. Upload `privacy-policy.html`.
3. In the repository, go to **Settings > Pages**.
4. Set the source to deploy from the default branch.
5. After GitHub Pages is enabled, your privacy policy URL will look like:

```text
https://YOUR_GITHUB_USERNAME.github.io/privacylens-policy/privacy-policy.html
```

Use that URL in Google Play Console as the Privacy Policy URL.
